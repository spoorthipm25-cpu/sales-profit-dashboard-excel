# 📊 Sales & Profit Dashboard (Excel Project)

## 📌 Overview
This project presents an interactive Excel dashboard built to analyze sales and profit performance for a supplement business across multiple dimensions.

## 🎯 Objective
To track key business KPIs and generate insights for better decision-making.

## 📊 Key Features
- KPI Tracking: Total Sales, Average Profit, Average Revenue, Units Sold
- Year-wise Profit Trend Analysis
- Location-wise Sales Distribution (Map)
- Platform-wise and Product-wise Performance Analysis
- Interactive slicers for dynamic filtering

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization Techniques

## 📈 Insights
- Identified top-performing platforms and products
- Observed profit trends across years
- Analyzed regional sales contribution
